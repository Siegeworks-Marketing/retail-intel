# Retail Intelligence (v1)

Daily retailer intelligence crawler and static reporting app.

## What this repo does
- Crawls reliable retailer sources (press rooms)
- Distills content into structured business events
- Writes versioned JSON data to the repo
- Publishes a static dashboard via GitHub Pages

## How it runs
- GitHub Actions (scheduled daily)
- Python for crawling and distillation
- Static HTML/JS for UI
